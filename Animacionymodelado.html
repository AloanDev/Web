Proyecto de Animación 3D

Realizado en Blender Cycles. Modelado y animación 3D. Texturizado a partir de cycles y con una iluminación muy resplandeciente. 

Procedimiento:

Modelado

Creé el robot y el entorno usando formas básicas como cilindros, esferas y planos, modificándolos con herramientas como extrusión, escalado y subdivisión para darles forma.

Materiales y Texturas

Apliqué materiales simples, utilizando shaders metálicos y de emisión para el cuerpo del robot y las luces, aprovechando Cycles para lograr un efecto de iluminación realista.

Iluminación

Coloqué luces en la escena (luces de área o de punto) para generar reflejos y sombras suaves, confiando en Cycles para obtener un realismo mayor con el trazado de rayos.

Cámara y Composición

Ubique la cámara en un ángulo que resalta la escena, creando una buena sensación de profundidad y un ambiente futurista.

Renderizado

Finalmente, configuré Cycles para utilizar el trazado de rayos, logrando un render con reflejos y sombras precisas.

Dinosaurios AR

En este proyecto realizado para ARDEV, creado con Unity y Blender, fue un concepto que desarrollamos para crear un juego de móviles.

Características:

AR para Moviles

Diseñado y desarrollado para que pudieras jugar en el móvil. La idea era poder ver dinosarios con la cámara del movil y poder atraparlos para coleccionarlos.

Unity y Vuforia

Para poder desarrollar dicho proyecto, utilizamos unos assets prestablecidos. Vuforia sería nuestra ayuda para poder crear este videojuego.

Interacción

Otra idea, era que los propios jugadores pudieran combatir con los dinosaurios ya capturados. Para ello, íbamos a desarrollar una arena donde poder combatir.
