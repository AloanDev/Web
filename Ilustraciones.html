Testimonio

Sancturaty Ep, fue diseñado con distintos tipos de pinceles, de los cuales, muchos tuve que improvisar y crear. Para ello, cree las texturas y las implemente como pinceles para ahorrar tiempo. Por ejemplo, si quería crear un tipo de hoja, creaba la hoja y gracias a la propia herramienta, Photoshop se encargaba de automatizar dicha hoja para que, al dibujar,
pintarade forma aleatoria o con algún rango predefinido.

- VECTORIZACIÓN -

Usando las herramientas de Adobe Photoshop e Ilustrator para
crear esta ilustración de forma que pareciera un modelo 3D Low Poly

  1. TABLETA Y PINCEL - Para crear estos banners publicitarios y contenido gráfico, utilice una tableta grafica para definir y perfeccionar el conjunto visual. Hay que tener en cuenta que es difícil sin esta herramienta, el hecho de retocar imágenes para que queden atractivas y visualmente bien.
  2. HERRAMIENTAS - Muchas de las herramientas utilizadas son, el tampón de clonar, distintos tipos de pincel, vectores, etc. Además, también opte por utilizar opciones de imagen como tono/saturación, niveles, filtros, curvas, etc. Estos utensilios me han servido para perfeccionar mi técnica como diseñador y ayudarme a crear contenido.
  3. SOLUCIONES DIFERENTES - Siempre he tenido que contar con varias respuestas para resolver cualquier problema con estas herramientas. Para ello, tuve el placer de aprender que no siempre hay un camino y que se puede crear contenido visual sin utilizar siempre la misma forma de hacerlo. ¡Hay que buscar soluciones!
